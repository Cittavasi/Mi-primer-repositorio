Las etiquetas de heading son utilizadas por lo motores de búsqueda para la identificación de títulos y subtítulos, por ello es importante no utilizarlas si sólo se requiere cambiar los estilos de los textos. Únicamente deben ser empleadas para la implementación de títulos y subtítulos

Para colocar los textos debe usarse la etiqueta de parrafo < p > < /p >

Para realizar separaciones de texto o secciones separadas se puede emplear la etiqueta < hr > esta etiqueta generará una línea de separación

Para dejar comentario en el código sin que aparezcan ene el navegador se emplea la siguiente etiqueta < ! --comentario-- >

La etiqueta < span > < /span > sirve para eqtuiquetar y referenciar texto dentro de la página web sin realizar ninguna modificación de estilo o estructural. Comunmente se emplea con la inteción de referenciar texto y después manipularlo con CSS o JS
